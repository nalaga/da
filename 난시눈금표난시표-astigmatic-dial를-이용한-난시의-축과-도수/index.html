  2. ⑤ 도수를정할때는..  보통 -0.25 D 혹은 -0.5 D 만큼씩 올려 …중략… 방향의 선이 똑같이 선명해보이는지 혹은 똑같이 흐려보이는지를 물어봅니다.
  3. 최종적으로 -1.25D Sph -1.0D Cyl Axis 90이 되었습니다.

앞선 포스팅을 통해서…

자각적 굴절검사의 종류에 대해서 알아보았습니다.

[안과 질환과 안과 치료/굴절광학]
자각적 굴절검사(주관적 굴절검사, subjective refraction)의 종류

자각적 굴절검사방법에는…

  1. 교차원주렌즈(cross cylinder)를 이용한 난시의 축과 도수 확인

  2. 난시눈금표(astigmatic dial)를 이용한 난시의 축과 도수 확인

  3. 적녹이색검사법(red green duochrome test)을 이용한 과교정여부 확인

  4. 양안 균형검사법(binocular balance test)을 이용한 양안 비교

이러한 방법들이 있습니다.

이중 난시의 축과 도수를 알수 있는 자각적 굴절검사에는..

교차원주렌즈(크로스실린더, cross cylinder) 방법과 난시눈금표(난시표, astigmatic dial)이 있습니다.

교차원주렌즈(크로스실린더) 방법에 대해서는 이전 포스팅을 참조하세요~

[안과 질환과 안과 치료/굴절광학] 교차원주렌즈(크로스실린더, Jackson cross cylinder, JCC)

교차원주렌즈(크로스실린더) 방법

이번에는… 난시눈금표(난시표, astigmatic dial)에 대해서 알아보겠습니다.





   


<난시 눈금표, 난시표(astigmatic dial)의 사용방법>

① 자동굴절검사계(AR) 혹은 검영법(MR) 을 이용하여,  우선 최대 교정시력이 나오도록 안경을 맞추어 끼웁니다.

예를들어서, -2.0D Sph 로, -2.0D 만큼의 근시를 가졌다고 해봅시다.

-2.0D Sph의 근시

② 이상태에서 기존의 안경에 +0.75D Sph의 구면렌즈(Spherical)를 가하여, 교정시력을 0.63 정도가 되도록 운무 (fogging)을 시행합니다.

이는 복합근시성 난시 상태로 만들어야만 결과해석에 용이하기 때문입니다.

복합근시성 난시 상태를 만들기 위해서 운무(fogging)

위의예인 -2.0D Sph 를 끼운 안경에서 +0.75D Sph 렌즈를 추가적으로 끼워준 상태에서 난시표를 보도록합니다.

그럼 최종적으로 -1.25 D Sph 의 안경을 끼운것과 동일합니다.

-2.0D Sph 안경에서 +0.75D Sph 렌즈를 추가적으로 끼워줌

-2.0D Sph 안경에 +0.75D Sph 렌즈를 끼워서 운무시킨 상태

③ 이렇게 운무시킨 상태에서 난시표를 보도록 합니다. 이때 어느방향의 선이 가장 진하게 보이는지를 물어봅니다.

난시눈금표(난시표, astigmatic dial)

④ 가장 진하게보인다고 대답한 선이 몇시 방향인지에 따라서 “30의 법칙(Rule of thirty)”에 따라서, 가장진하게보이는 방향의 시 * 30도를 하여 해당되는 방향에 (-) Cyl (마이너스 원주렌즈)를 끼워줍니다.

예를들어서 3시~9시 방향이 가장 진하다고 하면… 3시 * 30도 = 90도 방향에 (-) Cyl (마이너스 원주렌즈)를 끼웁니다.

3시~9시 방향이 가장 진하다고 하는 경우

6시~12시가 선명해보이면 180도에 끼우고, 3시~9시가 선명해보이면 90도방향에 끼우기

⑤ 보통 처음에는 -0.5D 의 (-) Cyl (마이너스 원주렌즈)를 끼웁니다.  

예시의 경우에서는….  교정용렌즈 -2.0D Sph 에 운무용렌즈 +0.75 DSph 를 끼운 기존 상태에서.. -0.5D Cyl Axis 90 을 끼워줍니다.

최종적으로 -1.25D Sph -0.5 D Cyl Axis 90이 됩니다.

-0.5D Cyl (마이너스 원주렌즈)를 끼워줌

-0.5D Cyl (마이너스 원주렌즈)을 90도에 맞춰서 끼워준 상태

이렇게 올리고 난다음에 다시 난시표를 보았을때, 진하게 보이는 선의 방향에 변화가 없는 경우 정확하게 교정된 것입니다.

그렇다면 모든 방향의 선이 똑같이 선명해 보일때까지 도수만 조정하면 됩니다.

만약 진해보이는 선의 방향에 변화가 있는 경우…

  1. 진하게 보이던 선의 방향이 반시계방향으로 이동하면, (-) Cyl (마이너스 원주렌즈)의 축방향을 시계방향으로 회전해 나갑니다.

  2. 진하게 보이던 선의 방향이 시계방향으로 이동하면, (-) Cyl (마이너스 원주렌즈)의 축방향을 반시계방향으로 회전해 나갑니다.

⑤ 도수를정할때는..  보통 -0.25 D 혹은 -0.5 D 만큼씩 올려가면서, 모든방향의 선이 똑같이 선명해보이는지 혹은 똑같이 흐려보이는지를 물어봅니다.

모든 방향에서 동일한 선명도로 보일때까지 도수를 올려줍니다.

위의 경우에서… -0.5D Cyl 에서 도수를 올려서 -1.0D Cyl 까지 올렸습니다.

최종적으로 -1.25D Sph -1.0D Cyl Axis 90이 되었습니다.

난시 도수를 -0.5DCyl 에서 -1.0D Cyl 로 올린 상태

⑥ 난시 도수가 결정되면..  운무법을 시행하기 위해 가해주었던 (+) Sph (플러스 구면렌즈)를 제거합니다.

운무법을 위해 가했던 +0.75D Sph 를 제거하여… -2.0D Sph -1.0D Cyl Axis 90이 된 모습입니다.

운무를 위해서 가했던 +0.75D Sph 를 제거한 상태

⑦ 마지막으로… 변화된 난시도수에 상응하는 구면렌즈대응치(Spherical equivalent)를 맞춰 주어야합니다.

초기 -2.0D Sph 에서  최종적으로 -2.0D Sph -1.0 D Cyl Axis 90으로  

난시도수가 -1.0D 만큼 변하였기에…  그절반만큼 구면렌즈를 반대 방향으로(+0.5 D) 바꾸어주어야 합니다.

그럼 최종적으로 처방하는 안경 도수는 -1.5D Sph -1.0DCyl Axis 90이 됩니다.

구면렌즈대응치(S.E)를 위해 -2.0D Sph 에서 -1.5D Sph 로 변경

최종적으로 처방하는 안경 도수는 -1.5D Sph -1.0DCyl Axis 90



   



난시표를 이용한 난시의 자각적인 측정은..


난시표에서 비교해야되는 여러선들의 선명도를 동시에 보면서 진행을 하므로, 더욱 진하게 보이느 선의 시계방향을 쉽게 찾아낼수 있다는 장점이 있지만..

난시표를 이용하는 경우, 난시표의 선이 30도 간격으로 되어 있거나, 폭이 좁아도 10도나 15도의 간격으로 되어 있어, 5도 단위로 처방하는 정확한 검사는 어려우며, 오차가 발생할 수 있습니다.
