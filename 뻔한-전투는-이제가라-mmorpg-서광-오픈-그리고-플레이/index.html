뻔한 전투는 이제가라??! MMORPG “서광” 오픈!!

오늘은 신규 오픈한 

MMORPG 서광

 대해서 소개하려고 해요.


 < 타임: 시간여행자 > 라는 모바일 게임을 만든 회사에서 

새롭게 출시한 MMORPG라고 하네요.

타임을 과금까지 하면서 했는데… 

그만큼 재미있었어요!!

그럼 서광을 플레이 하러 가볼까요??!!

<< MMORPG 서광 플레이하러가기 >>

게임을 다운 받고 실행시키니 

오늘 2시부터 정식서비를 시작했다고 

공지사항으로 알려주네요

현재 서버는 

S1. 아너 / S2. 지드 / S3. 야루스 

이 세개의 서버가 존재 하구요

저는 첫번째 서버인 아너를 선택했습니다.

그럼 플레이할 캐릭터를 둘러 볼까요?

나이트 계열에는 

워리어 / 글래디에이터 / 팔라딘이 있네요.

이렇게 디테일하게 케릭을 나눈 게임은 그렇게 흔치가 않죠?

레인져 케릭도 3가지로 나뉘는데요

샤프슈터 / 스카우트 / 위쳐

캐릭별로 입은 갑옷이 멋있는것 같아요.

서광에서 유일한 여자캐릭인

매지션 클래스 입니다.

매지션 / 프로즌 메이지 / 엘리멘탈리트

3개로 나뒤는데요. 

역시 여자 캐릭은 빵빵해야죠!! ㅋ

제가 플레이할 캐릭터는 

프로즌 메이지로 키우기위해 법사를 선택했습니다.

누가 그러드라구요.

“게임은 여자캐릭이 진리라고…”

믿거나 말거나.

처음시작하면 수녀와 이야기를 나누게 됩니다.

보시면 건너뛰기 버튼도 있구요.

터치진행 5초 가 써있는데 

5초가 지나면 자동으로 넘어가요.

건너뛰기 누르면 당연히 스킵!

위 사진전에 영상하나가 나오는데…

바로 이사진인데요.

조금 보다가 SKIP.

전체적인 인터페이스 인데요.

캐릭터 옆에 전투력이 표시되고

퀘스트를 누르면 바로 자동으로 진행합니다.

친절하게 누르라고 나오네요.

전반적인 인터페이스는 여타 게임과 다르지 않습니다.

좀 하다보면 이동하는게 나오는데 

용타고 이동하네요. 

이런건 그닥… ㅋ

당연히 출석 보상은 받아야겠죠??!!

일일접속 혜택은 시간이 지나면 하나씩 주니까 꼭 받기로 해요!

1레벨에 체험보상도 주고요.

어느정도 레벨이 오를때마다 체험보상을 주니 

꼭 받아서 사용하는게 좋아요.

그리고 업적도 있으니 꼭 완료한 업적은 

업적진도에 따라 아이템을 획득 할 수 있으니 놓치지 마시구요.

제캐릭이에요.. 섹쉬하죠??ㅋㅋ

어느덧 11렙이 됐구요!

말까지 타고 있답니다.

지금 레벨업을 할 수록

많은 아이템들을 주네요. 

사냥하는 플레이도 타격감이 괜찮아요~!!

이렇게 재미진 게임 서광!! 앞으로가 기대되는데요?!!

저도 열심히 키워보려구요 ^^

여러분도 지금 접속해서 시작해야 하지 않을까요?
