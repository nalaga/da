아파트 공시지가 조회 어떻게 하나요?

 

요즘 부동산 시장이 아주 난리죠? 정부에서는 부동산 투기 과열을 식힌다고 8.2 부동산 대책 이후 꾸준한 부동산 규제 정책을 발표하고 있잖아요. 특히 얼마 전 금리 인상 확정으로 인해 부동산 가격의 하락은 이제 피할 수 없게 되었는데요. 내 집마련을 생각하시는 분들에겐 희소식일 것이고, 부동산 투자를 위해 대출을 받아 집을 여러 채 보유하고 있는 분들에겐 안 좋은 소식일 것 같네요.

부동산 거래를 하시기 전엔 먼저 공시지가를 확인해 보실텐데요. 오늘은 부동산 공시지가를 확인하는 방법에 대해서 알아보도록 하겠습니다.

 


 

부동산 공시지가는 국토교통부의 인터넷 사이트를 통해 확인 하실 수 있는데요. 네이버나 다음과 같은 포털에서 [부동산공시가격알리미]라고 검색을 하시면 국토교통부에서 운영하는 홈페이지가 나올 겁니다. 해당 홈페이지로 접속해주세요.

 

첫 화면에 보시면 [공동주택 공시가격]이라는 아이콘이 보이실 텐데요. 해당 아이콘으로 들어가시면 됩니다. 그럼 아래와 같은 화면이 보일 텐데요.

 


 

위 페이지에서 해당 부동산/아파트의 주소를 도로명 주소 혹은 지번주소로 검색하신후 입력을 해주시면 됩니다. 아파트의 경우 만약 주소를 모르신다면 네이버에 해당 아파트를 검색하시면 주소가 나오니 참고해 주세요. 주소까지 모두 선택을 하신후, 세부주소까지 입력을 해주세요.

이제 끝입니다. 찾아보고자 하는 부동산/아파트의 송시기준, 동명, 호명, 단지명, 공동주택가격, 전용면적의 순서대로 확인 하실 수 있습니다. 아파트와 연립 다세대 주택과 같은 공동주택의 경우에는 국토교통부 장관이 매년 첫날인 1월 1일과 6월 1일에 공동주택의 적정가격을 고려하여 공동주택공시가격을 산정합니다.

지금까지 부동산/아파트의 공시지가 조회방법에 대해 알아보았습니다. 이제는 부동산/아파트의 공시지가 확인 위해 부동산을 직접 가지 않으시고, 인터넷 조회만으로 알아보실 수 있습니다.
