새로운 드래프트 픽


새로운 드래프트 픽 방식이 이번 패치로 전 지역

서버에 도입될 예정입니다.

일단 1월 13일부터 북미 / 터키 서버에 도입될 것이며

전 지역으로 확산해 나갈 생각입니다. 


포지션 정하기

로비에서 가고싶은 두 포지션을 정합니다.

-탑, 정글, 미드, 원딜, 서폿, 아무데나 (Fill)


포지션 확정

매치가 잡히면 자동적으로 각 플레이어에게 포지션을 정해줍니다.


픽 띄우기

자신의 팀원들에게 무슨 챔피언을 하고 싶은지 픽밴 전에 초상화를 띄워 보여줄 수 있습니다.


밴

각 팀의 3,4,5픽이 차례대로 3개의 밴을 합니다.


락 인

픽과 밴은 둘다 락인을 해야 합니다.

이제 초상화만 띄워놓는다고 픽 / 밴이 되지 않아요! 

(새로운 픽밴 시스템이 도입됨에 따라, 이제 팀 구성 모드는 사라지게 됩니다.)



2016시즌 랭크


이번 패치중으로 시작될 예정입니다. 


챔피언 변화


바드

이제 바드의 기본 공격 시, 정령은 타겟 반경 150도에 있는

나머지 유닛들에게도 피해를 입힙니다. 


브랜드

(버그픽스) 브랜드의 궁이 항상 적 챔피언을 향해 튕기는 버그를 수정했습니다.


케이틀린

(버그픽스) 케이틀린이 헤드샷 발동이 된 상대가

사거리 650 미만에 위치해있을 때

그것을 향해 걸어가는 버그를 수정했습니다.


헤드샷 다음의 평타가 평소보다 느리게 나가는 버그를 수정했습니다.


케이틀린이 헤드샷을 쏘기 전에 타깃을 바꿀 때 딜레이가 생기는 버그를 수정했습니다.


피들스틱

어둠의 바람 (e) 스킬의 마나 소모량이 50/70/90/110/130 ⇒ 50/60/70/80/90 로 감소했습니다.


갱플랭크

패시브 지속시간 : 1.5초 -> 2.5초 로 증가


틱 (ticks) 데미지를 입는 횟수로 보시는게 편할듯 싶네요

6회 -> 10회 (틱당 입는 데미지량 감소, 전체 데미지는 변화 없음)


계수 : 1.2 추가 공격력 -> 1.0 추가 공격력

(버그픽스) 징크스가 억제기를 파괴할 시 패시브스택을 하나 이상으로 얻는 버그를 수정했습니다.


Q, 로켓을 마나 없이 쏠 수 있는 버그를 수정했습니다.


징크스의 패시브로부터 얻는 공속이 로켓 런쳐에 제대로 적용되지 않는 버그를 수정했습니다.

(의도된 공속보다 적게 적용됐습니다.)


이제 Q가 비활성화 되어 있을 시, 사용할 수 없습니다.

뽑아 찢기 (e) 마나 소모량 : 40 -> 30

뽑아 찢기로 둘 이상의 유닛을 처치 했을 시 받는 마나량 : 40 -> 30

총알은 비를 타고 (e) 데미지가 90/145/200/255/310 ⇒ 80/115/150/185/220 로 감소.

미니언 변화


더 이상 미니언이 중간에 멈추지 않습니다.

이제 미니언들의 추가 데미지량과 데미지 감소량을 볼 수 있습니다. 

더 이상 미니언들은 정글이나 강가에서 새로운 타깃을 얻지 않습니다.

미니언이 정글이나 강가에 있을 시 벗어나기 전 까지

새로운 타깃을 얻지 않습니다.

주위의 미니언이 공격 당했을 시 도움을 요청합니다 (?)


정글


새로운 정글 아이템 : 룬의 메아리


재료 :

업그레이드 된 정글 아이템 + 증폭의 고서 + 에테르 환영 + 조합비 340 g


총 비용 : 2625 골드

주문력 : 60

이동속도 : 10%

고유지속효과 : 이동 , 혹은 공격시 스택을 얻습니다. 스택이 꽉 찼을 시, 80 + (0.10 주문력)의 마법 피해를 4명의 적에게 입힙니다. 대형 몬스터들에겐 250%의 피해를 입힙니다. 또한, 대형 몬스터를 가격 했을 시, 잃은 마나의 18%를 돌려받습니다. 


강타

예를 들면 주문 흡혈이나, 라일라이 지속 효과를 발동시키지 않음.

이제 상대방의 와드로부터 900내의 범위일 때에 발동 됩니다. (전엔 1200)

이제 상대방의 파란 장신구 와드는 500내의 범위일 때에 발동 됩니다. (전엔 1200)
