나눔경영컨설팅 ‘김종혁’대표 ( 마케팅, 동기부여, 협상, 소통, PT&기획 교육전문 ) 

안녕하십니까? 

저는 대기업과 중견기업에서 해외영업,주재원(중국,대만),전략기획담당등 10년이상의 현업경력과 고려대 경영전문대학원 (경영학석사/마케팅전공) 졸업의 최종학력을 보유하고 있으며, 매번 새로운 Case와 실제경험으로 다양하고 재미있는 교육을 진행하여, 수강생의 평가도 굉장히 좋은 나눔경영컨설팅 대표 ‘김종혁강사’입니다. 유튜브채널에서 실제 강의영상을 확인하실 수 있습니다. 

https://www.youtube.com/c/educasttv  

컨설턴트 자격증, 교사 자격증, MBTI 지도사 자격증등을 보유하고 있으며, 강의경력은 7년차이고 주로 B2B마케팅, 해외신규개척전략, 중국이문화의 이해, SNS바이럴마케팅, 협상, 기획관련하여 중소벤처기업부, 테크노파크, 고용노동부  및 대기업(삼성,현대,LG등)과 연계하여 총1,200여회 이상의 교육을 진행해왔습니다. 특기는 3개국어소통(영어,일어,중어)에 재즈피아노 연주, 캐리커쳐, 카툰, 동영상 촬영&편집입니다. (특기를 활용하여 강의를 흥미롭게 진행하고 있습니다.) 

주요사항은 아래와 같으며 상세이력은 첨부파일 참조하시기 바랍니다. 

1. Value : FUN, Performance, Convenience / 교육생들께는 ‘재미’와 ‘퍼포먼스’를, 교육담당자들께는 ‘편의성’의 가치를 제공하겠습니다.

2. 강의 방법 : 강의식( 애플 Key note, 프레지, 동영상 사용) / 능숙한 PT / 동영상&음악 맞춤편집/ 맥북와 아이팟 무선연결 / 음악,미술,카툰활용/ 음악프로그램 ‘로직’와 건반악기를 이용한 연주시연 / ‘파이널컷프로’를 활용한 동영상 편집시연

3. 주요Topic : 30개국의 해외경험과 10년간의 치열한 해외영업, 주재원 경험 및 실제 창업경험을 통한 조직의 소통,위기극복,B2B마케팅,창업&취업전략및 SNS활용,기획과 프레젠테이션에 대하여 강의 (세부 강의주제는 첨부파일을 참조하시기 바랍니다.)
