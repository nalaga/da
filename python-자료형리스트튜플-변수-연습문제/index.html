[연습문제]

1. 초를 입력 받아서 시간과 분으로 변환하는 프로그램

 

  – shell에서 아닌 editor에서 소스를 작성했다. 일단 내가 초를 입력받으면 한번에 분과 시간을 한번에 출력되는걸 원했기 때문이다.

  – 일단 sec는 int 형변환 하는 이유는 타입이 str이기 때문에 int 바꾼 후에 계산을 했다.

  – 소스를 작성한 후 F5 shell에 실행해봤다.

 

  – editor을 shell에 실행하게 된다면 이런 화면이 나온다. 초 : 이곳에 숫자를 작성하게 되면 바로 분과 시간이 바뀌게 된다.

 

  – 초에 내가 원하는 숫자를 작성한 후 enter를 누르면 바로 분과 시간이 계산되어 나타난다.

2. 섭씨를 입력받아서 화씨로 변환하는 프로그램
   F = 9.0 / 5.0 * C + 32

 

  – 1번 문제와 비슷하게 shell이 아닌 editor에 소스를 작성한 후 shell에 실행을 했다.

  – 일단 섭씨는 정수이기 때문에 int로 형변환을 했고, 화씨는 정수가 아닌 다른 수도 나올수 있기 때문에 float로 형변환을 했다.

 

  – editor을 shell에 실행하게 된다면 이런 화면이 나온다. 섭씨 : 이곳에 정수를 작성하게 되면 바로 계산되어 화씨가 출력이 된다.

 

3. 화씨를 입력받아서 섭씨로 변환하는 프로그램
   C = (5 * F – 160) / 9.0

 

  – 연습문제 2번과 비슷하게 소스를 작성했다. 거의 같지만 다른건 화씨가 정수 아닌 수가 나올수 있기 때문에 float로 형변환을 했다.

  – 화씨에서 섭씨를 입력받을때는 정수가 아닌 다른 수가 나올 확률이 있으므로 int가 아닌 float로 형변환을 했다.

 

  – editor을 shell에 실행하게 된다면 이런 화면이 나온다. 화씨 : 이곳에 수를 작성하게 되면 바로 계산되어 섭씨가 출력이 된다.
