(구매 링크) ASUS ROG Maximus X Hero (Wi-Fi AC) LGA1151 DDR4 DP HDMI M.2 Z370 ATX Motherboard with onboard 802.11ac WiFi, Gigabit LAN and USB 3.1 for 8th Generation Intel Core Processors

 

ASUS의 메인보드는 언제나 가성비 최고에 명품이라는 호칭이 따라 붙는 것 같습니다. 커뮤니티를 보면 언제나 ASUS나 ASROCK 중에서 고민하시는 분들이 많은 거 같아요. ASROCK이 오버를 좀 더 잘 받쳐준다고는 하는데 오버쿨럭 이슈는 딱히 ASUS에서도 안나왔고, 성능 상에 큰 차이는 일반적으로 느껴지지 않습니다.

 

ASUS의 마더보드 시리즈는 많지만, 이번은 Z370 MAXIMUS X HERO 입니다. 조금 고가의 장비이지만, 컴퓨터 튠을 하시는 분들이면 직구로 많이들 구매합니다.

 

아마존 구매 시 박스 구성품은 아래와 같습니다.

 

뭐, 기본적인 것은 다 들어 있고 ROG 스티커를 통해 커스터마이징이 되는 것도 언제나 흥미로운 거 같아요.

 

 

제품의 상세한 설명을 조금 해드릴게요.

 

ASUS ROG MAXIMUS X HERO는 오버클러커들의 사랑을 흠껏받아온 막시무스(maximus) 시리즈의 10세대 제품이고, 카피레이크 시대에 맞게 새로이 출시되었죠. 이 상위 호환될 모델은 FORMULA, CODE, APEX 등 모델도 있지만 베스트셀러이자 가성비 최고인 막시무스는 언제나 사랑받습니다.

 

ASUS는 TPU, EPU, DIGI+전원부, FAN EXPERT 4, 터보 어플이 조합된 5-WAY 최적화가 적용되어 안정성과 오버쿨럭에 대한 내구를 높였고, 핵사/쿼드 프로세서인 커피레이크 프로세서에 ‘마이크로 파인 ALLOY’ 초크와 넷플렉스 모스펫 10K 캐퍼시터를 구성하여 안정적인 전력 공급으로 전력 문제 해결에도 신경을 썼습니다.

더욱이, 우리가 CPU를 볼 때 가장 중요시 보는게 계속 언급한 ‘오버 쿨러킹’인데, 베이스 쿨럭은 PRO Clock ll와 전원/셋업 버튼과 원터치 clear cmos/bios 플래시백 버튼, Dr.Debug LED와 Q-LED 를 제공하여 효율적인 오버쿨러킹을 돕고 있습니다.

 

 

 

메모리는 최대 2,666MHz DDR4 지원하고, 최대 4,133MHz 메모리 쿨럭까지 발휘할 수 있습니다. 인텔 옵테인 메모리 PCIe 3.0 x4 m.2 슬롯을 갖추었고, 3개의 M.2 SSD를 장착한다고 함은 하이퍼 튜닝에도 적합한 마더보드임을 증명합니다.

 

여기에 6개의 SATA 3 Port + USB 2.0, 4개의 USB 3.1 Gen 1포트 등, 다양한 포트와 많은 패널용 포트를 갖추어 폭넓은 장비의 지원이 가능합니다.

 

다음은 가격입니다.

 

 

현재 아마존에서는 ASUS Maximus X HERO가 273.93$이고, 재고가 있으며 국내 직배송까지 됩니다.

현재 디파짓이 44.31$로 걸려있어서 합치면 약 320$, 한화로 약 34~35만원이 나옵니다. 디파짓이 있으면 관부가세도 디파짓으로 환산을 하기때문에 구매만 하시면 별 걱정없이 받아 볼 수 있으며, 아마존 새 제품 발송은 언제나 a/s가 편리합니다.

 

단, 관부가세와 배송비가 44.31$라면 제 기준으로는 적합하기때문에 별도의 배대지를 찾기 보다는 그냥 구매하겠지만, 자신이 배대지를 잘 알고 쿠폰이 있다(오레곤이나 디오 등에 거점을 둔 배대지는 더욱 저렴)면 배대지를 이용하시면 됩니다. 제 생각에는 배대지를 쓰나 아마존을 쓰나 몇 달러 차이는 안날 것으로 사료됩니다.

 

 

국내 유통가는 최소 42만원~보통 45만원~최대53만원 선으로 표시가 됩니다. 근데 여기 업체들도 해외직구를 통해 들여오는 제품이라 재고에 따라 오히려 해외 직구하시는 것보다 더 오래 걸릴 수 있으며, 지금 가격만 비교해보아도 아마존 대비 5~10만원 더 비쌉니다.

 

따라서, 아마존 직구로 ASUS MAXIMUS X HERO 메인보드를 5~10만원 더 저렴하게 구매하시길 바랍니다. 앞으로도 메인보드를 좀 더 올려야겠네요. 올리는 동안 저도 많은 정보를 얻었고, 흥미로운 것 같습니다.

 

링크 남겨드릴테니 참고하세요.
