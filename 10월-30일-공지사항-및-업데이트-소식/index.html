“부호의 문” 출현!

▲ FFBE 정식 서비스 기념 첫번째 이벤트 “부호의 문” 개방

▲ 10월 30일 정기점검 후 ~ 11월 2일 23시59분 59초

▲ 스테미나 5, 10, 15 소모의 총 3개의 던전으로 구성, 일반 퀘스트 맵보다 “길”을 많이 드랍

   ※ 맵 클리어 시 라피스 지급 없음



교환소 추가

▲ 스타쿼츠를 통해 아이템 교환 가능  

▲ 교환 장소 : 이리의 이빨(雪狼の牙) 탐색맵을 통해 개방한 마를로의 숨겨진 마을의 뚱뚱한 초코보에게 아이템 교환 가능

                                                                 ☞ 숨겨진 마을 찾는 법 링크

▲ 교환 가능 품목

1. 피닉스의 꼬리 : 부활 1회 / 스타쿼츠 5개

2. 치유의 지팡이 : 공격+11, 마력+5, 정신+26 / 스타쿼츠 10개

= 착용 : 케알가(초급 전체 회복마법) 사용 가능

3.검은갑옷 : 공  격+5, 방어+35 / 스타쿼츠 20개

4. 빛의 팔찌 : 스타쿼츠 30개

= 착용 : 매턴 시작 시 프로테스(방어 마법) 시전

인터페이스

▲ 홈화면 및 메뉴 아이콘에 미 수령 훈장 개수 표시

▲ 파티편성 화면에서 장비 일괄 해제 및 파티 정보 보기 기능 추가 

▲ 탐색 퀘스트 진행중 아이템을 이용해 상태이상 회복 가능 

▲ 친구 헬퍼 유닛 사용 후, 일정 시간 뒤 재사용 시 인연 포인트 재 지급

▲ 공지사항 아이콘 변경



버그 수정 

▲ 강제종료 버그 수정

▲ 어빌리티 “난사” 사용 시 강제종료 되는 현상 수정

▲ 연결 상태 확인 아이콘 수정

▲ 친구 헬퍼 선택 화면에서 환수 및 장비 표시가 올바르지 않던 현상 수정

▲ 퀘스트 보상 화면에서 이전 등급에서 습득한 마법/어빌이 다시 표시 되던 현상 수정
