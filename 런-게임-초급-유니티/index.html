들어가며

 일단 초급 강의부터 듣고 와야 겠다… 다짜고짜 중급강의를 들으려고 하니까… 힘이 드는 듯… 우선 Brackeys님의 초보용 강의를 들을 생각이다… 상속과 관련된 고급 문법은 천천히 익히자…

강의



 미리 한번 흟고 왔는데… 이건 내가 할수 있겠다는 생각이 든다… 이것도 도앙치면 그냥 유니티는 접어야 하지 않을 까 싶다… 강의마다 링크를 퍼오기가 이제 좀 귀찮아서 소개 부분만 퍼왔다…



1강

 Shift + Space로 게임뷰를 키웠다가 줄였다가 할수가 있다… 평소 게임 뷰류를 작게 하고 씬뷰를 크게 놔두기 때문에 유용한 기능인듯… 스카이 박스를 바꾸는 건 카메라에 가서 Clear Flags 손보고 Background 에서 바꿀 수 있다… 줌은 F키고 Shift + 휠키 누르기가 화면이동… 마우스 왼쪽키가 회전이다… 화면에 보여지는게 Mash고 외곽 감지가 Collider인듯…

 기초적인거 배울게 있는데… 나름 게임 엔진 만저봤다고 초급 강의 건너 뛴건 오판인듯… 처음 부터 다시 시작하는 마음으로 강의를 차근 차근 진행해야 겠다…

2강

 스크립트 쪽은 새로 배우는게 없어서… 패스… 

 하루에 3강의 정도가 적당한듯 싶다… 


3강

 Physics Material 이라는 새로운 개념을 배웠다… Collider에 등록되는 거인듯… 이거 이분 다른 강의들도 들으면서 초급 개념을 좀 익혀야 할듯 싶다… 

 고닷엔진에 대한 기억이 슬슬 가물 가물해져가는데… 내기억엔 FixedUpdate나 Input 관련해서 Time.deltaTime을 까먹지 않고 쓰도록 매개변수에 표시 해두지 않았나 싶다… 둘의 경우에는 deltaTime이 필수이니 말이다…

using UnityEngine;
using UnityEngine.SceneManagement;

public class GameManager : MonoBehaviour {

    bool gameHasEnded = false;

    public float restartDelay = 1f;

    public void EndGame () {

        if (gameHasEnded == false)
        {
            gameHasEnded = true;
            Debug.Log(“Game Over”);
            Invoke(“Restart”, restartDelay);
        }
        
    }

    void Restart () {
        SceneManager.LoadScene(SceneManager.GetActiveScene().name);
    }
}

 머리 식히는 맘으로 가볍게 보고 있는 중이다…

    [Range(.100f, 1000f)]
    public float forwardForce = 500f;

    public float sidewaysForce = 50f;


 [Range (최소, 최대)]를 통해서 인스펙터 창에 바를 만들수 있는 듯 싶다… 그 플레이 모드시 색을 바꾸게 하는 것도 꿀팁인듯…


    public new Rigidbody rigidbody;

    [Space]

    [Header (“Add force”)]
    [Range(.100f, 1000f)]
    public float forwardForce = 500f;
    [Range(.10f, 100f)]
    public float sidewaysForce = 50f;


 이것도 상당한 꿀팁이다…


    public new Rigidbody rigidbody;

    [Space]

    [Header (“Add force”)]

    [SerializeField]
    [Range(.100f, 1000f)]
    float forwardForce = 500f;
    [SerializeField]
    [Range(.10f, 100f)]
    float sidewaysForce = 50f;


 이건 더 미친 꿀팁인데… public을 선언을 안해도 인스펙터 창에서 편집을 할수가 있다… public은 가급적 선언을 안해주는게 실수를 막을 수 있다는 점에서 미친 꿀팁인듯…
