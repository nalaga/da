Linear and Logistic Regression in TensorFlow¶

1. Linear Regression in TensorFlow¶

  • 보험회사는 화재, 강도사건의 빈도를 예측의 피쳐로 사용함
  • 이를 어떻게 함수로 알수 있는가? Y = f(X)
  • 데이터는 시카고의 화재강도 사건빈도임
  • X = fires per 1000 housing units
  • Y = thefts per 1000 population
  • 우리는 특정 지역의 화재사고 건수로 강도사건의 발생을 예측하려함
  • 화재와 강도 사이에는 어떤 관계가 있는가? => Y = wX + b
  • 로스함수로 에러제곱평균을 구해서 w와 b의 스칼라 파라미터을 알아내야함

코드를 분석해 보자¶

  • 처음 텐서 플로우 코드를 접하면 다음 2가지 이유로 매우 혼동된다.
  • 1) Why is train_op in the fetches list of tf.Session.run()?
  • 2) How does TensorFlow know what variables to update?
  • 학습 오퍼레이션은 로스를 최소화하는목적을 가지며, 로스는 w와 b에 의존한다.
  • 즉, 그래프의 마지막 학습세션에 tf.Session.run()이 작동하면 결국 옵티마이저는 loss를 가져다 쓰고, loss를 계산하기 위해서는 w와 b가 필요하기때문에 전체 그래프가 모두 작동되게 됨
    • optimizer = tf.train.GradientDescentOptimizer(learning_rate=0.01).minimize(loss)
    • sess.run(optimizer, feed_dict={X: x, Y:y})

Optimizers¶

  • GradientDescentOptimizer는 그라디언트 디센트룰에 따라 업데이트함을 의미한다
  • 텐서플로우는 우리를 위해 자동으로 미분을하여 로스를 최소화하는 값을 업데이트한다
  • 디폴트 상황에서 옵티마이저는 모든 변수를 학습하지만,
  • 특정 변수를 학습하지 않게 하려면 trainable = False 인수를 사용한다.

  • 5,000 points of validation data (MNIST.validation)

X, Y에 대한 플레이스 홀더 생성¶

Step 1: 데이터 로드¶

Step 2: 파라미터 정의¶

  • 러닝레이트 : 가중치를 조정하는 파라미터
  • 배치사이즈 : 전체 데이터를 학습하기 위해 몇개로 분할해 놓은것을 배치하고함
  • 에포치 : 한번의 배치를 학습하면 1에포치가 됨

Step 3: 피쳐와 라벨에 대한 플레이스 홀더 생성¶

  • 인풋 : 28*28 사이즈 수열
  • 아웃풋 : 0-9사이의 손글씨 10종

Step 4 : 가중치와 바이어스 변수 생성¶

  • W : 정규분포, 평균0, 표준편차 0.01의 분포에서 랜덤값 할당
  • b : 초기값을 0으로 할당
  • W의 차원은 Y에 depends함 = [784, 10] / Y = tf.matmul(X, w) 계산을 위해서..
  • b의 차원은 Y에 depends함 = [1,10]

Step 5 : 예측모델 정의¶

  • W*x + b 를 행렬곱으로 계산

Step 6 : 로스 함수 정의¶

  • 소프트맥스 함수의 로스인 크로스엔트로피 계산
  • 크로스엔트로피를 최소화하는 로스펑션 정의

import time
import numpy as np
import tensorflow as tf
from tensorflow.examples.tutorials.mnist import input_data

# Step 1: Read in data
MNIST = input_data.read_data_sets("/data/mnist", one_hot=True)

# Step 2: Define parameters for the model
learning_rate = 0.01
batch_size = 128
n_epochs = 25

# Step 3: create placeholders for features and labels
X = tf.placeholder(tf.float32, [batch_size, 784])
Y = tf.placeholder(tf.float32, [batch_size, 10])

# Step 4: create weights and bias
w = tf.Variable(tf.random_normal(shape=[784, 10], stddev=0.01), name="weights")
b = tf.Variable(tf.zeros([1, 10]), name="bias")

# Step 5: predict Y from X and w, b
logits = tf.matmul(X, w) + b

# Step 6: define loss function
# use softmax cross entropy with logits as the loss function
# compute mean cross entropy, softmax is applied internally
entropy = tf.nn.softmax_cross_entropy_with_logits(logits, Y)
loss = tf.reduce_mean(entropy) # computes the mean over examples in the batch

# Step 7: define training op
# using gradient descent with learning rate of 0.01 to minimize cost
optimizer = tf.train.GradientDescentOptimizer(learning_rate=learning_rate).minimize(loss)

init = tf.initialize_all_variables()

with tf.Session() as sess:
    sess.run(init)
    n_batches = int(MNIST.train.num_examples/batch_size)  #429개 배치
    for i in range(n_epochs): # train the model n_epochs times
        for _ in range(n_batches):
            X_batch, Y_batch = MNIST.train.next_batch(batch_size)
            sess.run([optimizer, loss], feed_dict={X: X_batch, Y:Y_batch})

# test the model            
    n_batches = int(MNIST.test.num_examples/batch_size)  #78개 배치
    total_correct_preds = 0
    for i in range(n_batches):
        X_batch, Y_batch = MNIST.test.next_batch(batch_size)
        _, loss_batch, logits_batch = sess.run([optimizer, loss, logits],feed_dict={X: X_batch, Y:Y_batch})        
        preds = tf.nn.softmax(logits_batch)
        correct_preds = tf.equal(tf.argmax(preds, 1), tf.argmax(Y_batch, 1))
        accuracy = tf.reduce_sum(tf.cast(correct_preds, tf.float32)) 
        total_correct_preds += sess.run(accuracy)
    print "Accuracy {0}".format(total_correct_preds/MNIST.test.num_examples)
